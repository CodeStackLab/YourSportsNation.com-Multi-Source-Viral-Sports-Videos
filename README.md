# YourSportsNation.com — Multi-Source Viral Sports Videos | YouTube API + NewsAPI + Dailymotion + Reddit + RSS

## Description
This repository contains the completely automated viral sports video aggregation and publishing workflow for the **YourSportsNation.com** project. It leverages multiple APIs, social media platforms, and data sources to curate high-quality sports content completely on autopilot.

## Features
- **YouTube API Integration:** Automatically fetches trending sports highlights and related content.
- **NewsAPI Integration:** Aggregates real-time sports news, headlines, and updates.
- **Dailymotion API:** Supplements video content with alternative sources.
- **Reddit API:** Pulls engaged discussions, clips, and top posts from sports communities.
- **RSS Feeds:** Continuously monitors various sports blogs and websites for the latest updates.
- **Automated Workflow:** Specifically designed to run as an automated workflow file that manages polling from multiple sources, processing the data, and executing automated posts.
